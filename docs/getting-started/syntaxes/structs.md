# Structs

In Crabby, The `struct` keyword is a **custom data type** that allows you to group together related data under a single name.

Example:

```rs
struct Data {
    x: Int,
    y: Int,
}

pub struct Person {
    name: String,
    age: Int,
    hobbies: String,
    birthday: Date,
}
```
