# Loops

In Crabby, you can do loops by using the `loop` & `for-in` keywords.

```rs
loop 100 {
    print("Hello, World!")
}
```

```py
for i in range(10) {
    print(i)
}
```

You can also use the `range` keyword just like in Python:

```py
let x = 200

for i in range(x) {
    print(i)
}
```
